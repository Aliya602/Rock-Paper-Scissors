# Rock-Paper-Scissors
A Rock Paper Scissors game created in Python  with series mode, score tracking, and colourful output on command line

## Features
- 🎮 Play against the computer
- 📍 Series mode - first to win target rounds wins
- 📊 Score tracking throughout
- 👤 Personalized with player name
- 🎨 Colorful output with colorama
- ✅ Input validation and error handlin

## How to play
1. Run the script (command line)
2. Enter your name
3. Choose series mode or normal mode
4. Enter rock, paper, or scissors
5. Try to beat the computer!

## How to run the code
Run Python code

## Requirement
- Python 3.x
- colorama (pip install colorama)
